Description

Creates an attribute rule to populate symbol rotation based on intersecting line features.

Usage

          



License
Standard

Syntax

SymbolRotation (out_folder, in_table, line_classes, in_field, {where_clause}, {in_subtypes}, {orientation}, {rotation_style}, {additional_angle}, {type}, {is_editable}, {triggering_events}, {error_number}, {error_message}, {description}, {exclude_from_client_evaluation}, {batch}, {severity}, {tags})

Parameter Explanation Data Type
out_folder Dialog Reference

The folder where the output CSV will be saved. The name of the file is taken from in_table.

 		DEFolder
in_table Dialog Reference

The table or feature class that will have the new rule applied.

 		DEFeatureClass
line_classes Dialog Reference

The line feature classes to intersect. The numeric Orientation Field is used as a tie-breaker when there are multiple intersecting lines.

 		GPValueTable
in_field Dialog Reference

The name of an existing field to which the rule will be applied.

 		Field
where_clause (Optional) Dialog Reference

SQL Expression to filter records. Only records that satisfy the query will be calculated.

 		GPSQLExpression
in_subtypes (Optional) Dialog Reference

The subtypes to which the rule will be applied.

 		GPString
orientation (Optional) Dialog Reference

Defines which feature to orient towards when there are multiple intersecting lines.

  • Checked - Use the higher field value. This is the default.
  • Unchecked - Use the lower field value.

 		GPBoolean
rotation_style (Optional) Dialog Reference

Defines how direction is measured.

  • Arithmetic - Rotation starts at zero in the east direction and rotation is counterclockwise.
  • Geographic - Rotation starts at zero in the north direction and rotation is calculated clockwise. This is the default.

 		GPString
additional_angle (Optional) Dialog Reference

The additional amount to add to the calculated value. Defaults to 0.

 		GPDouble
type (Optional) Dialog Reference

Specifies the type of attribute rule that will be added.

 		GPString
is_editable (Optional) GPBoolean
triggering_events (Optional) GPString
error_number (Optional) GPLong
error_message (Optional) GPString
description (Optional) GPString
exclude_from_client_evaluation (Optional) GPBoolean
batch (Optional) GPBoolean
severity (Optional) GPLong
tags (Optional) GPString
